# スプライン補間 学習・計算ツール

## 概要

本Webアプリケーションは、**三次スプライン補間**を手軽に試せる計算ツールであり、同時に補間計算の仕組みを学べる教材ツールです。

### 主な特徴

- **2種類の補間モード**に対応
  - **通常補間（関数型）**: Y = f(X) の形で、X が単調増加している場合に対応
  - **パラメトリック補間**: X = x(t), Y = y(t) の形で、ループ状や X が単調増加でない曲線に対応

- **学習用機能**
  - 補間計算に用いられた行列、係数、区間ごとの補間式を表示
  - スプライン補間の数学的背景を説明
  - 小規模データ（3～5点）では詳細な中間計算結果を表示

- **使いやすいUI**
  - CSV/TSV形式での点群テキスト貼り付け入力
  - グラフによる補間結果の可視化（元点と補間曲線を区別表示）
  - 補間後点群のワンクリックコピー

## 使用方法

### 起動

1. `index.html` をWebブラウザで開く
2. または、ローカルサーバーで serve する場合：
   ```bash
   python -m http.server 8000
   # または
   npx http-server
   ```
   その後、 `http://localhost:8000/spline_webapp/` にアクセス

### 基本的な操作フロー

1. **補間モードの選択**
   - 「通常補間」または「パラメトリック補間」を選択

2. **点群データの入力**
   - テキストボックスに点を入力（カンマ区切りまたはタブ区切り）
   - 形式例：
     ```
     0,0
     1,2
     2,1
     3,3
     ```

3. **分割数の指定**
   - 各区間を何分割するかを指定（デフォルト: 10）
   - 値を大きくするほど、より多くの補間点が生成される

4. **実行**
   - 「実行」ボタンをクリック
   - グラフが表示され、補間結果が出力される

5. **結果の確認**
   - グラフで補間曲線を視覚的確認
   - 計算過程で行列、係数、各区間の補間式を確認

6. **データのコピー**
   - 「コピー」ボタンで補間後の点群をクリップボードにコピー

### サンプルデータ

「サンプル入力」ボタンで、プリセットされたサンプルデータを読み込めます。

## 技術仕様

### 実装言語

- **フロントエンド**: HTML5, CSS3, JavaScript (Vanilla)
- **計算エンジン**: JavaScript（行列演算、ガウス・ジョルダン法）
- **グラフ描画**: Chart.js v3.9.1
- **数式表示**: MathJax 3

### スプライン補間アルゴリズム

#### 通常補間（関数型）

各区間 $k$ で以下の3次式を定義：
$$S_k(h) = a_k h^3 + b_k h^2 + c_k h + d_k$$

ここで $h = \frac{x - x_k}{\Delta h_k}$ （0 ≤ h ≤ 1）

#### 連続条件

隣接区間での連続性：
- **位置連続**: $a_k + b_k + c_k = y_{k+1} - y_k$
- **1階導関数連続**: $\frac{3a_k + 2b_k + c_k}{\Delta h_k} = \frac{c_{k+1}}{\Delta h_{k+1}}$
- **2階導関数連続**: $\frac{6a_k + 2b_k}{\Delta h_k^2} = \frac{2b_{k+1}}{\Delta h_{k+1}^2}$

#### 境界条件（自然スプライン）

$$b_0 = 0, \quad \frac{6a_{n-1} + 2b_{n-1}}{\Delta h_{n-1}^2} = 0$$

#### 線形システム

上記の条件から $3n \times 3n$ の線形システムを構築し、ガウス・ジョルダン法で解く。

#### パラメトリック補間

パラメータ $t$ を導入し、$x(t)$ と $y(t)$ をそれぞれ独立してスプライン補間。
初期段階では等間隔パラメータを採用。

### 行列計算

- **逆行列**: ガウス・ジョルダン法（ピボット選択付き）
- **行列乗算**: 標準的なO(n³)アルゴリズム
- 数値安定性: 近似値による計算のため、丸め誤差が蓄積する場合あり

## 入力チェック

以下の検証を実施：

- ✓ 点数が3点以上であること
- ✓ 各行が2列（X, Y）であること
- ✓ X, Y が数値であること
- ✓ 通常補間モードで X が単調増加していることを警告
- ✓ 同一 X 値の重複を警告
- ✓ 分割数が1以上であること

## 学習用機能

### 表示される情報

#### 小規模データ（点数 ≤ 5）の場合

- 区間幅（Δh）一覧
- 各区間の係数（a, b, c）
- 各区間のスプライン式（詳細形）

#### 大規模データの場合

- 区間幅（Δh）の統計情報
- 係数の省略表示

### 数式表示

MathJax により、以下が LaTeX 形式で表示：

$$S_k(h) = a_k h^3 + b_k h^2 + c_k h + d_k$$

## トラブルシューティング

### グラフが表示されない

- JavaScript コンソール（F12）でエラーを確認
- Chart.js が正しく読み込まれているか確認（CDN接続）

### 計算エラーが出る

- 入力データに不正な値がないか確認
- 点数が3点以上であることを確認
- 数値の精度（特に非常に大きい/小さい値）を確認

### 式が正しく表示されない

- MathJax が正しく読み込まれているか確認
- ブラウザキャッシュをクリア

## VBA 実装との対応

本アプリケーションは、Excel VBA の `NaturalSplineInterp()` 関数を JavaScript に移植しています。

### 対応部分

- ✓ 区間幅の計算（式(1)）
- ✓ 正規化パラメータ（式(2)）
- ✓ スプライン式（式(3)）
- ✓ 補間条件（式(5)）
- ✓ 1階導関数連続条件（式(6)）
- ✓ 2階導関数連続条件（式(7)）
- ✓ 自然スプライン境界条件（式(8)）

### 追加実装

- パラメトリック補間（VBA未実装）
- UI・グラフ表示（Webアプリケーション固有）
